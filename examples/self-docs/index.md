# Diataxis Documentation Skill

Documentation that lives outside a structured workflow drifts: tutorials rot, reference docs contradict the code, and revision feedback gets lost between regeneration cycles. The Diataxis Documentation Skill gives Claude Code a repeatable six-step process — scope, structure, generate, score, revise, build — so that documentation stays accurate and internally consistent without constant manual oversight.

**Audience**: Developers and technical writers using Claude Code
