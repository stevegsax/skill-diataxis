How-to guides are task-focused recipes for readers who already know what
they want to do. They assume basic competence and skip teaching.

- [How to Install and Set Up](install-and-setup/) — install `uv`, Hugo
  extended, Go, `make`, and `marimo`.
- [How to Build and Serve Documentation](build-and-serve/) — `make build`,
  `make serve`, theme switching, and deploying.
- [How to Score and Revise Documentation](score-and-revise/) — run the
  deterministic check suite, then ask Claude for the qualitative scoring
  pass.
