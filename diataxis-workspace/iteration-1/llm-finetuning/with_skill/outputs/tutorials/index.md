# Tutorials

These tutorials guide you through fine-tuning an LLM on your local machine, step by step. Work through them in order -- each one builds on the previous.

1. [Environment Setup](environment-setup.md) -- Install Python, PyTorch, and the Hugging Face ecosystem
2. [Dataset Preparation](dataset-preparation.md) -- Load, format, and tokenize training data
3. [Configure QLoRA Training](configure-qlora-training.md) -- Set up quantized model loading, LoRA adapters, and training arguments
4. [Run Training](run-training.md) -- Execute the training run and monitor progress
5. [Evaluate Model](evaluate-model.md) -- Compare outputs, score quality, and merge the adapter

Each tutorial includes an interactive exercise notebook for hands-on practice.
