# Tutorial: Your First Steps with Fractions

In this tutorial, we will build a solid foundation with fractions. By the end, you
will be able to read and write fractions, identify their parts, create equivalent
fractions, simplify fractions to their lowest terms, and work with mixed numbers.

## What we will need

- A willingness to work through each step (no skipping ahead)
- Comfort with basic multiplication and division of whole numbers

## Step 1: The parts of a fraction

Let's start with a simple fraction:

```
  3
 ---
  4
```

Every fraction has two parts:

- **Numerator** (top number): tells us how many parts we have. Here, **3**.
- **Denominator** (bottom number): tells us how many equal parts make up the whole. Here, **4**.

We read this as "three-fourths" or "three quarters."

**Check your understanding**: In the fraction 5/8, the numerator is **5** and the
denominator is **8**. We have 5 parts out of 8 equal parts.

Let's do one more: the fraction 7/10. The numerator is **7** and the denominator is
**10**. We read this as "seven-tenths" --- we have 7 parts out of 10 equal parts.

## Step 2: Reading fractions from a visual model

Imagine a pizza cut into 4 equal slices. You eat 1 slice.

```
  [ X ] [   ] [   ] [   ]
   ate   left   left  left
```

The fraction of pizza you ate is **1/4** (one-fourth).
The fraction remaining is **3/4** (three-fourths).

Now imagine a pizza cut into 8 equal slices. You eat 3 slices.

```
  [ X ] [ X ] [ X ] [   ] [   ] [   ] [   ] [   ]
```

The fraction you ate is **3/8**. The fraction remaining is **5/8**.

Let's try a measuring cup. You have a 1-cup measure divided into 3 equal sections.
You fill it to the second mark:

```
  |===|===|   |
   1/3  2/3  3/3
```

You have **2/3** of a cup. The numerator 2 tells us how many sections are filled.
The denominator 3 tells us the cup is divided into 3 equal sections.

**Notice that** the denominator always matches the total number of equal pieces,
and the numerator counts how many pieces we are talking about.

## Step 3: Equivalent fractions

Let's go back to our pizza. Take a pizza cut into 4 slices and eat 2 of them:

```
  [ X ] [ X ] [   ] [   ]       -->  2/4
```

Now imagine cutting each of those 4 slices in half, giving us 8 slices total.
The amount of pizza eaten has not changed, but now we can describe it differently:

```
  [X] [X] [X] [X] [ ] [ ] [ ] [ ]   -->  4/8
```

We have just discovered that **2/4 = 4/8**. These are **equivalent fractions** ---
different ways of writing the same amount.

**The rule**: We can create an equivalent fraction by multiplying (or dividing)
both the numerator and denominator by the same number.

Let's verify with 2/4. We multiply both parts by 2:

```
2/4  -->  (2 x 2) / (4 x 2)  =  4/8    Correct.
```

Try another: starting from 1/3, multiply both parts by 4:

```
1/3  -->  (1 x 4) / (3 x 4)  =  4/12
```

So **1/3 = 4/12**. We multiplied the top and bottom by the same number (4),
keeping the value identical.

One more: starting from 3/5, multiply both parts by 3:

```
3/5  -->  (3 x 3) / (5 x 3)  =  9/15
```

So **3/5 = 9/15**. We can verify by going the other direction: divide both parts
of 9/15 by 3, and we get back to 3/5.

## Step 4: Simplifying fractions

Simplifying means finding the equivalent fraction with the smallest possible
numbers. We do this by dividing both parts by their greatest common factor (GCF).

Let's simplify **6/8**:

1. Find the GCF of 6 and 8. Factors of 6: 1, 2, 3, 6. Factors of 8: 1, 2, 4, 8.
   The greatest common factor is **2**.

2. Divide both parts by 2:

```
6/8  -->  (6 / 2) / (8 / 2)  =  3/4
```

The simplified form of 6/8 is **3/4**.

Let's try another: simplify **10/15**.

1. GCF of 10 and 15: factors of 10 are 1, 2, 5, 10; factors of 15 are 1, 3, 5, 15.
   GCF is **5**.

2. Divide both by 5:

```
10/15  -->  (10 / 5) / (15 / 5)  =  2/3
```

You should see that **10/15 = 2/3**.

One more: simplify **12/18**.

1. GCF of 12 and 18: factors of 12 are 1, 2, 3, 4, 6, 12; factors of 18 are
   1, 2, 3, 6, 9, 18. GCF is **6**.

2. Divide both by 6:

```
12/18  -->  (12 / 6) / (18 / 6)  =  2/3
```

So **12/18 = 2/3**.

## Step 5: Mixed numbers and improper fractions

So far, all our fractions have been less than 1 (the numerator was smaller than
the denominator). But what happens when we have more than a whole?

Suppose you eat 5 slices of a pizza that was cut into 4 slices (you grabbed a
slice from a second pizza). That's **5/4** --- an **improper fraction**, where
the numerator is larger than the denominator.

We can also write this as a **mixed number**: **1 1/4**. That means 1 whole pizza
plus 1 extra quarter.

### Converting an improper fraction to a mixed number

Let's convert **7/3** to a mixed number.

1. Divide the numerator by the denominator: 7 / 3 = **2** remainder **1**.
2. The quotient (2) becomes the whole number part.
3. The remainder (1) becomes the new numerator. The denominator stays the same (3).

```
7/3  =  2 remainder 1  =  2 1/3
```

**Result**: 7/3 = 2 1/3.

Let's do another: convert **11/4** to a mixed number.

1. Divide: 11 / 4 = **2** remainder **3**.
2. Whole number part: 2. New numerator: 3. Denominator stays: 4.

```
11/4  =  2 remainder 3  =  2 3/4
```

**Result**: 11/4 = 2 3/4.

### Converting a mixed number to an improper fraction

Let's convert **3 2/5** to an improper fraction.

1. Multiply the whole number by the denominator: 3 x 5 = **15**.
2. Add the numerator: 15 + 2 = **17**.
3. Keep the same denominator: 5.

```
3 2/5  =  (3 x 5 + 2) / 5  =  17/5
```

**Result**: 3 2/5 = 17/5.

One more: convert **2 3/8** to an improper fraction.

1. Multiply: 2 x 8 = **16**.
2. Add numerator: 16 + 3 = **19**.
3. Keep denominator: 8.

```
2 3/8  =  (2 x 8 + 3) / 8  =  19/8
```

**Result**: 2 3/8 = 19/8.

## What we accomplished

In this tutorial, we:

- Identified the numerator and denominator as the two parts of every fraction
- Read and wrote fractions from visual models (pizza slices, measuring cups)
- Created equivalent fractions by multiplying both parts by the same number
- Simplified fractions to lowest terms by dividing both parts by their GCF
- Learned about mixed numbers and improper fractions
- Converted 7/3 to 2 1/3 and 11/4 to 2 3/4 (improper to mixed)
- Converted 3 2/5 to 17/5 and 2 3/8 to 19/8 (mixed to improper)
