# Tutorial: Your First Steps with Fractions

In this tutorial, we will build a solid foundation with fractions. By the end, you
will be able to read and write fractions, identify their parts, create equivalent
fractions, and simplify fractions to their lowest terms.

## What we will need

- A willingness to work through each step (no skipping ahead)
- Comfort with basic multiplication and division of whole numbers

## Step 1: The parts of a fraction

Let's start with a simple fraction:

```
  3
 ---
  4
```

Every fraction has two parts:

- **Numerator** (top number): tells us how many parts we have. Here, **3**.
- **Denominator** (bottom number): tells us how many equal parts make up the whole. Here, **4**.

We read this as "three-fourths" or "three quarters."

**Check your understanding**: In the fraction 5/8, the numerator is **5** and the
denominator is **8**. We have 5 parts out of 8 equal parts.

## Step 2: Reading fractions from a visual model

Imagine a pizza cut into 4 equal slices. You eat 1 slice.

```
  [ X ] [   ] [   ] [   ]
   ate   left   left  left
```

The fraction of pizza you ate is **1/4** (one-fourth).
The fraction remaining is **3/4** (three-fourths).

Now imagine a pizza cut into 8 equal slices. You eat 3 slices.

```
  [ X ] [ X ] [ X ] [   ] [   ] [   ] [   ] [   ]
```

The fraction you ate is **3/8**. The fraction remaining is **5/8**.

**Notice that** the denominator always matches the total number of equal pieces,
and the numerator counts how many pieces we are talking about.

## Step 3: Equivalent fractions

Let's go back to our pizza. Take a pizza cut into 4 slices and eat 2 of them:

```
  [ X ] [ X ] [   ] [   ]       -->  2/4
```

Now imagine cutting each of those 4 slices in half, giving us 8 slices total.
The amount of pizza eaten has not changed, but now we can describe it differently:

```
  [X] [X] [X] [X] [ ] [ ] [ ] [ ]   -->  4/8
```

We have just discovered that **2/4 = 4/8**. These are **equivalent fractions** ---
different ways of writing the same amount.

**The rule**: We can create an equivalent fraction by multiplying (or dividing)
both the numerator and denominator by the same number.

Let's verify:

```
2/4  -->  (2 x 2) / (4 x 2)  =  4/8    Correct.
```

Try another: starting from 1/3, multiply both parts by 4:

```
1/3  -->  (1 x 4) / (3 x 4)  =  4/12
```

So **1/3 = 4/12**. You should see that we multiplied the top and bottom by the
same number (4), keeping the value identical.

## Step 4: Simplifying fractions

Simplifying means finding the equivalent fraction with the smallest possible
numbers. We do this by dividing both parts by their greatest common factor (GCF).

Let's simplify **6/8**:

1. Find the GCF of 6 and 8. Factors of 6: 1, 2, 3, 6. Factors of 8: 1, 2, 4, 8.
   The greatest common factor is **2**.

2. Divide both parts by 2:

```
6/8  -->  (6 / 2) / (8 / 2)  =  3/4
```

The simplified form of 6/8 is **3/4**.

Let's try another: simplify **10/15**.

1. GCF of 10 and 15: factors of 10 are 1, 2, 5, 10; factors of 15 are 1, 3, 5, 15.
   GCF is **5**.

2. Divide both by 5:

```
10/15  -->  (10 / 5) / (15 / 5)  =  2/3
```

You should see that **10/15 = 2/3**.

## What we accomplished

In this tutorial, we:

- Identified the numerator and denominator as the two parts of every fraction
- Read and wrote fractions from visual models
- Created equivalent fractions by multiplying both parts by the same number
- Simplified fractions to lowest terms by dividing both parts by their GCF
