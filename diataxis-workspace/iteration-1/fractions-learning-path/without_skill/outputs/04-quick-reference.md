# Fractions Quick Reference Card

Keep this handy while working through the exercises.

---

## Key Definitions

    Numerator / Denominator = Fraction

    Proper:   numerator < denominator       (e.g. 3/8)
    Improper: numerator >= denominator       (e.g. 7/4)
    Mixed:    whole number + proper fraction (e.g. 1 3/4)

---

## Core Operations

### Simplify

    Divide top and bottom by their GCD.
    12/18 --> GCD is 6 --> 2/3

### Add / Subtract

    Same denominator:    a/n +- b/n = (a +- b)/n
    Different denominator: find LCD, convert, then add/subtract.

### Multiply

    a/b * c/d = (a*c) / (b*d)
    Tip: cross-cancel common factors first.

### Divide

    a/b / c/d = a/b * d/c    (flip the second fraction)

---

## Conversions

### Mixed --> Improper

    whole * denominator + numerator
    -----------------------------------
              denominator

### Improper --> Mixed

    Divide numerator by denominator.
    Quotient = whole number, remainder = new numerator.

### Fraction --> Decimal

    Divide numerator by denominator.

### Decimal --> Fraction

    Write over the appropriate power of 10, then simplify.
    0.75 = 75/100 = 3/4

---

## Finding LCD (Least Common Denominator)

    List multiples of each denominator.
    The smallest number appearing in both lists is the LCD.

    Denominators 4 and 6:
    4: 4, 8, 12, 16 ...
    6: 6, 12, 18 ...
    LCD = 12

## Common Fraction/Decimal/Percent Equivalences

    1/2 = 0.5   = 50%       1/5 = 0.2  = 20%
    1/3 = 0.333 = 33.3%     1/6 = 0.167 = 16.7%
    1/4 = 0.25  = 25%       1/8 = 0.125 = 12.5%
    3/4 = 0.75  = 75%       1/10 = 0.1  = 10%
    2/3 = 0.667 = 66.7%     3/8 = 0.375 = 37.5%
