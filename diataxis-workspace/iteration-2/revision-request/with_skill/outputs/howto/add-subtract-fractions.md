# How to Add and Subtract Fractions

This guide shows you how to add and subtract fractions in common real-world
scenarios.

## How to add fractions with different denominators

**Goal**: Compute the sum of two fractions that do not share a denominator.

1. Find the least common denominator (LCD) of the two denominators.
    - List multiples of each denominator until you find the smallest shared one.
    - Example: for 1/3 + 1/4, multiples of 3 are 3, 6, 9, **12**; multiples of
      4 are 4, 8, **12**. LCD = 12.

2. Convert each fraction to an equivalent fraction with the LCD.
    - 1/3 = 4/12 (multiply top and bottom by 4)
    - 1/4 = 3/12 (multiply top and bottom by 3)

3. Add the numerators. Keep the denominator.
    - 4/12 + 3/12 = 7/12

4. Simplify if possible.
    - 7/12 is already in lowest terms.

**Result**: 1/3 + 1/4 = 7/12

## How to subtract a fraction from a whole number

**Goal**: Compute a whole number minus a fraction (e.g., 3 - 1/4).

1. Write the whole number as a fraction with denominator 1.
    - 3 = 3/1

2. Find the LCD of 1 and the fraction's denominator.
    - LCD of 1 and 4 is 4.

3. Convert the whole number fraction.
    - 3/1 = 12/4

4. Subtract the numerators.
    - 12/4 - 1/4 = 11/4

5. Convert to a mixed number if desired.
    - 11/4 = 2 3/4

**Result**: 3 - 1/4 = 2 3/4

## How to add mixed numbers that require carrying

**Goal**: Add two mixed numbers where the fractional parts sum to more than 1
(e.g., 2 3/4 + 1 2/3).

1. Add the whole number parts.
    - 2 + 1 = 3

2. Add the fractional parts (find LCD if needed).
    - 3/4 + 2/3: LCD = 12
    - 9/12 + 8/12 = 17/12

3. If the fraction sum is improper, convert it.
    - 17/12 = 1 5/12

4. Add the extra whole number to the whole number sum.
    - 3 + 1 5/12 = 4 5/12

5. Simplify the fraction if possible.
    - 5/12 is already in lowest terms.

**Result**: 2 3/4 + 1 2/3 = 4 5/12
