# Fraction Operations Reference

## Terminology

| Term | Definition | Example |
|------|-----------|---------|
| Fraction | A number expressed as one integer divided by another | 3/4 |
| Numerator | The top number; how many parts are taken | 3 in 3/4 |
| Denominator | The bottom number; how many equal parts make the whole | 4 in 3/4 |
| Proper fraction | Numerator is less than denominator | 3/4 |
| Improper fraction | Numerator is greater than or equal to denominator | 7/4 |
| Mixed number | A whole number combined with a proper fraction | 1 3/4 |
| Reciprocal | A fraction flipped: numerator and denominator swap | Reciprocal of 3/4 is 4/3 |
| Lowest terms | Numerator and denominator share no common factor other than 1 | 3/4 (not 6/8) |
| LCD | Least Common Denominator; smallest number that is a multiple of both denominators | LCD of 3 and 4 is 12 |
| GCF | Greatest Common Factor; largest number that divides both numerator and denominator | GCF of 6 and 8 is 2 |

## Equivalent Fractions

| Rule | Formula | Example |
|------|---------|---------|
| Multiply both parts by the same nonzero number | a/b = (a*n)/(b*n) | 1/3 = 2/6 = 3/9 |
| Divide both parts by the same nonzero number | a/b = (a/n)/(b/n) | 6/8 = 3/4 |

## Simplifying Fractions

| Step | Action | Example (12/18) |
|------|--------|-----------------|
| 1 | Find the GCF of numerator and denominator | GCF(12, 18) = 6 |
| 2 | Divide both by the GCF | 12/6 = 2, 18/6 = 3 |
| 3 | Result is in lowest terms | 2/3 |

## Converting Between Improper Fractions and Mixed Numbers

| Direction | Procedure | Example |
|-----------|-----------|---------|
| Improper to mixed | Divide numerator by denominator. Quotient = whole part, remainder = new numerator. | 7/4 = 1 remainder 3 = 1 3/4 |
| Mixed to improper | Multiply whole number by denominator, add numerator. Keep same denominator. | 2 1/3 = (2*3 + 1)/3 = 7/3 |

## Addition and Subtraction

| Case | Procedure | Example |
|------|-----------|---------|
| Same denominator | Add/subtract numerators, keep denominator | 2/5 + 1/5 = 3/5 |
| Different denominators | Find LCD, convert both fractions, then add/subtract | 1/3 + 1/4 = 4/12 + 3/12 = 7/12 |
| Mixed numbers | Convert to improper fractions (or add whole and fraction parts separately) | 1 1/2 + 2 1/3 = 3/2 + 7/3 = 9/6 + 14/6 = 23/6 = 3 5/6 |

## Multiplication

| Case | Procedure | Example |
|------|-----------|---------|
| Two fractions | Multiply numerators, multiply denominators | 2/3 * 4/5 = 8/15 |
| Fraction and whole number | Write whole number as fraction over 1, then multiply | 3 * 2/5 = 3/1 * 2/5 = 6/5 |
| Mixed numbers | Convert to improper fractions first, then multiply | 1 1/2 * 2/3 = 3/2 * 2/3 = 6/6 = 1 |
| Cross-cancellation (optional) | Before multiplying, divide any numerator and any denominator by a common factor | 2/3 * 3/4: cancel 3s = 2/1 * 1/4 = 2/4 = 1/2 |

## Division

| Case | Procedure | Example |
|------|-----------|---------|
| Two fractions | Multiply by the reciprocal of the divisor | 2/3 / 4/5 = 2/3 * 5/4 = 10/12 = 5/6 |
| Fraction by whole number | Write whole number as fraction, then invert and multiply | 3/4 / 2 = 3/4 * 1/2 = 3/8 |
| Mixed numbers | Convert to improper fractions first, then invert and multiply | 1 1/2 / 3/4 = 3/2 * 4/3 = 12/6 = 2 |
