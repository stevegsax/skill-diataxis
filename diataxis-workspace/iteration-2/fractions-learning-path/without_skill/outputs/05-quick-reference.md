# Fractions Quick Reference

A one-page cheat sheet for the core operations.

## Simplify

Divide numerator and denominator by their GCD.

## Add / Subtract

```
a/b + c/d = (a*d + c*b) / (b*d)     then simplify
a/b - c/d = (a*d - c*b) / (b*d)     then simplify
```

Or find the LCD first to keep numbers smaller.

## Multiply

```
a/b x c/d = (a*c) / (b*d)           then simplify
```

Cross-cancel common factors before multiplying.

## Divide

```
a/b / c/d = a/b x d/c               ("Keep, Change, Flip")
```

## Mixed Number to Improper Fraction

```
W n/d = (W*d + n) / d
```

## Improper Fraction to Mixed Number

```
Divide numerator by denominator.
Quotient = whole part, remainder = new numerator.
```

## Fraction to Decimal

```
Divide numerator by denominator.
```

## Decimal to Fraction

```
Write as [digits] / [power of 10], then simplify.
0.75 = 75/100 = 3/4
```

## Comparing Fractions

Cross-multiply: compare a*d with b*c for a/b vs c/d.

## Common Equivalences

```
1/2 = 0.5        1/3 = 0.333...     1/4 = 0.25
1/5 = 0.2        1/6 = 0.1666...    1/8 = 0.125
2/3 = 0.666...   3/4 = 0.75         3/8 = 0.375
5/8 = 0.625      7/8 = 0.875
```
