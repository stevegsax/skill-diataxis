# Fractions Refresher for Adults

This guide covers fractions from the ground up, designed for adults revisiting high school math. Each section builds on the previous one.

## What Is a Fraction?

A fraction represents a part of a whole. It is written as two numbers separated by a horizontal bar:

```
  numerator
 -----------
 denominator
```

- The **numerator** (top) tells you how many parts you have.
- The **denominator** (bottom) tells you how many equal parts the whole is divided into.

Example: 3/4 means "3 out of 4 equal parts."

## Types of Fractions

- **Proper fraction** -- the numerator is smaller than the denominator (e.g., 2/5, 7/8). Its value is less than 1.
- **Improper fraction** -- the numerator is equal to or larger than the denominator (e.g., 5/3, 9/4). Its value is 1 or greater.
- **Mixed number** -- a whole number combined with a proper fraction (e.g., 1 2/3, 2 1/4).

### Converting Between Improper Fractions and Mixed Numbers

**Improper fraction to mixed number:**

1. Divide the numerator by the denominator.
2. The quotient is the whole number part.
3. The remainder becomes the new numerator; the denominator stays the same.

Example: 11/4 --> 11 divided by 4 = 2 remainder 3 --> 2 3/4

**Mixed number to improper fraction:**

1. Multiply the whole number by the denominator.
2. Add the numerator.
3. Place the result over the original denominator.

Example: 3 2/5 --> (3 x 5 + 2) / 5 = 17/5

## Equivalent Fractions

Two fractions are equivalent if they represent the same value. You create equivalent fractions by multiplying or dividing both the numerator and denominator by the same nonzero number.

```
1/2 = 2/4 = 3/6 = 4/8 = 50/100
```

This is the fundamental principle behind simplifying fractions and finding common denominators.

## Simplifying (Reducing) Fractions

To simplify a fraction, divide both the numerator and denominator by their greatest common divisor (GCD).

Example: Simplify 18/24

1. Factors of 18: 1, 2, 3, 6, 9, 18
2. Factors of 24: 1, 2, 3, 4, 6, 8, 12, 24
3. GCD = 6
4. 18/24 = (18 / 6) / (24 / 6) = 3/4

## Adding and Subtracting Fractions

### Same Denominator

Add or subtract the numerators. Keep the denominator.

```
2/7 + 3/7 = 5/7
5/9 - 1/9 = 4/9
```

### Different Denominators

1. Find the least common denominator (LCD) -- the smallest number both denominators divide into evenly.
2. Rewrite each fraction with the LCD.
3. Add or subtract the numerators.
4. Simplify if possible.

Example: 1/3 + 1/4

1. LCD of 3 and 4 = 12
2. 1/3 = 4/12, 1/4 = 3/12
3. 4/12 + 3/12 = 7/12

Example: 5/6 - 1/4

1. LCD of 6 and 4 = 12
2. 5/6 = 10/12, 1/4 = 3/12
3. 10/12 - 3/12 = 7/12

## Multiplying Fractions

Multiply numerator by numerator and denominator by denominator.

```
2/3 x 4/5 = (2 x 4) / (3 x 5) = 8/15
```

Tip: You can cross-cancel before multiplying to keep numbers small.

```
4/9 x 3/8
```

The 4 and 8 share a factor of 4 (4/4 = 1, 8/4 = 2). The 3 and 9 share a factor of 3 (3/3 = 1, 9/3 = 3).

```
1/3 x 1/2 = 1/6
```

## Dividing Fractions

To divide by a fraction, multiply by its reciprocal (flip the second fraction).

```
2/3 / 4/5 = 2/3 x 5/4 = 10/12 = 5/6
```

Memory aid: "Keep, Change, Flip" -- keep the first fraction, change division to multiplication, flip the second fraction.

## Fractions and Decimals

To convert a fraction to a decimal, divide the numerator by the denominator.

```
3/4 = 3 / 4 = 0.75
1/3 = 1 / 3 = 0.333...
```

To convert a terminating decimal to a fraction, use place value:

```
0.625 = 625/1000 = 5/8
```

## Comparing Fractions

To compare fractions with different denominators, rewrite them with a common denominator, then compare numerators.

Example: Which is larger, 3/5 or 2/3?

```
3/5 = 9/15
2/3 = 10/15
```

Since 10/15 > 9/15, we know 2/3 > 3/5.

Cross-multiplication shortcut: compare 3 x 3 = 9 with 5 x 2 = 10. Since 10 > 9, the second fraction (2/3) is larger.

## Order of Operations with Fractions

Fractions follow the same PEMDAS/BODMAS rules as whole numbers:

1. Parentheses / Brackets
2. Exponents / Orders
3. Multiplication and Division (left to right)
4. Addition and Subtraction (left to right)

Example: 1/2 + 3/4 x 2/3

1. Multiplication first: 3/4 x 2/3 = 6/12 = 1/2
2. Then addition: 1/2 + 1/2 = 1
