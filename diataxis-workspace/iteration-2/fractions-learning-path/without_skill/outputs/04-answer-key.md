# Answer Key

Detailed solutions for all exercises in 03-exercises.md.

## Set A: Fundamentals

**1.** 24/36 = 2/3
GCD of 24 and 36 is 12. Divide both by 12.

**2.** 45/60 = 3/4
GCD of 45 and 60 is 15. Divide both by 15.

**3.** 23/5 = 4 3/5
23 / 5 = 4 remainder 3.

**4.** 7 3/8 = 59/8
7 x 8 + 3 = 59.

**5.** Equivalent fractions to 2/7: 4/14, 6/21, 8/28 (multiply numerator and denominator by 2, 3, or 4).

**6.** 5/11 > 4/9
Cross-multiply: 4 x 11 = 44, 9 x 5 = 45. Since 45 > 44, 5/11 is larger.

## Set B: Addition and Subtraction

**7.** 3/8 + 5/8 = 8/8 = 1

**8.** 7/10 - 3/10 = 4/10 = 2/5

**9.** 1/6 + 3/8
LCD = 24. 1/6 = 4/24, 3/8 = 9/24. Sum = 13/24.

**10.** 5/9 + 7/12
LCD = 36. 5/9 = 20/36, 7/12 = 21/36. Sum = 41/36 = 1 5/36.

**11.** 7/8 - 2/3
LCD = 24. 7/8 = 21/24, 2/3 = 16/24. Difference = 5/24.

**12.** 4 2/5 + 3 1/3
Convert: 22/5 + 10/3. LCD = 15. 66/15 + 50/15 = 116/15 = 7 11/15.

**13.** 6 1/4 - 2 5/6
Convert: 25/4 - 17/6. LCD = 12. 75/12 - 34/12 = 41/12 = 3 5/12.

**14.** 3/7 + 2/5 - 1/2
LCD = 70. 3/7 = 30/70, 2/5 = 28/70, 1/2 = 35/70.
30/70 + 28/70 - 35/70 = 23/70.

## Set C: Multiplication and Division

**15.** 3/4 x 2/5 = 6/20 = 3/10

**16.** 7/9 x 3/14
Cross-cancel: 7 and 14 (factor 7), 3 and 9 (factor 3).
= 1/3 x 1/2 = 1/6.

**17.** 5/6 x 12/25
Cross-cancel: 5 and 25 (factor 5), 12 and 6 (factor 6).
= 1/1 x 2/5 = 2/5.

**18.** 2 1/3 x 1 4/5
Convert: 7/3 x 9/5. Cross-cancel 9 and 3 (factor 3).
= 7/1 x 3/5 = 21/5 = 4 1/5.

**19.** 5/8 / 3/4 = 5/8 x 4/3
Cross-cancel 4 and 8 (factor 4).
= 5/2 x 1/3 = 5/6.

**20.** 9/10 / 3/5 = 9/10 x 5/3
Cross-cancel 9 and 3 (factor 3), 5 and 10 (factor 5).
= 3/2 x 1/1 = 3/2 = 1 1/2.

**21.** 4 1/2 / 2 1/4 = 9/2 / 9/4 = 9/2 x 4/9
Cross-cancel 9s, then 4 and 2 (factor 2).
= 1/1 x 2/1 = 2.

**22.** 7/12 / 7/6 = 7/12 x 6/7
Cross-cancel 7s, then 6 and 12 (factor 6).
= 1/2.

## Set D: Mixed Operations and Order of Operations

**23.** 1/2 + 1/3 x 3/4
Multiplication first: 1/3 x 3/4 = 3/12 = 1/4.
Then: 1/2 + 1/4 = 2/4 + 1/4 = 3/4.

**24.** (2/3 + 1/6) x 4/5
Parentheses first: 2/3 + 1/6 = 4/6 + 1/6 = 5/6.
Then: 5/6 x 4/5 = 20/30 = 2/3.

**25.** 5/8 - 1/4 / 2
Division first: 1/4 / 2 = 1/4 x 1/2 = 1/8.
Then: 5/8 - 1/8 = 4/8 = 1/2.

**26.** (3/4 - 1/3) / (1/2 + 1/6)
Numerator: 3/4 - 1/3 = 9/12 - 4/12 = 5/12.
Denominator: 1/2 + 1/6 = 3/6 + 1/6 = 4/6 = 2/3.
Division: 5/12 / 2/3 = 5/12 x 3/2 = 15/24 = 5/8.

**27.** 2/3 x 3/4 + 1/6 x 2/3
First product: 2/3 x 3/4 = 6/12 = 1/2.
Second product: 1/6 x 2/3 = 2/18 = 1/9.
Sum: 1/2 + 1/9 = 9/18 + 2/18 = 11/18.

## Set E: Conversions

**30.** 0.375 = 375/1000 = 3/8

**31.** 0.8333... Let x = 0.8333...

## Set F: Word Problems

**33.** 3 3/4 - 1 2/3
= 15/4 - 5/3. LCD = 12.
= 45/12 - 20/12 = 25/12 = 2 1/12 feet.

**34.** The added water fills 7/10 - 2/5 = 7/10 - 4/10 = 3/10 of the tank.
3/10 of the capacity = 15 gallons.
Total capacity = 15 / (3/10) = 15 x 10/3 = 50 gallons.

**35.** First person: 1/3. Second person: 3/7.
Combined: 1/3 + 3/7 = 7/21 + 9/21 = 16/21.
Third person: 1 - 16/21 = 5/21.
Amount: 5/21 x $84 = $420/21 = $20.

**36.** 2 1/4 + 1 5/8 + 3 1/2
= 9/4 + 13/8 + 7/2. LCD = 8.
= 18/8 + 13/8 + 28/8 = 59/8 = 7 3/8 miles.

**37.** Scaling factor: 10/4 = 5/2.
Sugar: 3/4 x 5/2 = 15/8 = 1 7/8 cups.

**38.** Together per hour: 1/6 + 1/4 = 2/12 + 3/12 = 5/12 of the pool.
Time to fill: 1 / (5/12) = 12/5 = 2 2/5 hours (2 hours and 24 minutes).
