# Fraction Terminology Reference

## Core Definitions

| Term | Definition | Example |
|------|-----------|---------|
| Fraction | A number of the form $\frac{a}{b}$ where $b \neq 0$ | $\frac{3}{4}$ |
| Numerator | The top number in a fraction; the count of parts | In $\frac{3}{4}$, the numerator is $3$ |
| Denominator | The bottom number in a fraction; the number of equal parts in the whole | In $\frac{3}{4}$, the denominator is $4$ |
| Proper fraction | A fraction where the numerator is less than the denominator; value is between $0$ and $1$ | $\frac{3}{4}$, $\frac{1}{2}$ |
| Improper fraction | A fraction where the numerator is greater than or equal to the denominator; value is $\geq 1$ | $\frac{7}{4}$, $\frac{5}{3}$ |
| Mixed number | A whole number combined with a proper fraction | $1\frac{3}{4}$, $2\frac{1}{3}$ |
| Reciprocal | The fraction obtained by swapping numerator and denominator | Reciprocal of $\frac{3}{4}$ is $\frac{4}{3}$ |
| Equivalent fractions | Fractions that represent the same value | $\frac{1}{2} = \frac{2}{4} = \frac{3}{6}$ |
| Lowest terms | A fraction where the GCF of numerator and denominator is $1$ | $\frac{3}{4}$ is in lowest terms; $\frac{6}{8}$ is not |
| GCF (Greatest Common Factor) | The largest positive integer that divides both numbers | GCF of $12$ and $18$ is $6$ |
| LCD (Least Common Denominator) | The smallest positive integer that is a multiple of all given denominators | LCD of $\frac{1}{3}$ and $\frac{1}{4}$ is $12$ |
| LCM (Least Common Multiple) | The smallest positive integer that is a multiple of both numbers. LCD is the LCM of the denominators. | LCM of $4$ and $6$ is $12$ |

## Notation Conventions

| Notation | Meaning |
|----------|---------|
| $\frac{a}{b}$ | Fraction with numerator $a$ and denominator $b$ |
| $a\frac{b}{c}$ | Mixed number: $a + \frac{b}{c}$ |
| $a \div b$ or $\frac{a}{b}$ | Division of $a$ by $b$ |

## Equivalent Fraction Property

$$\frac{a}{b} = \frac{a \times k}{b \times k} \quad \text{for any } k \neq 0$$
