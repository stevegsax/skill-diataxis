# Fraction Addition and Subtraction Rules

## Like Denominators

| Operation | Rule | Example |
|-----------|------|---------|
| Addition | $\frac{a}{c} + \frac{b}{c} = \frac{a + b}{c}$ | $\frac{2}{7} + \frac{3}{7} = \frac{5}{7}$ |
| Subtraction | $\frac{a}{c} - \frac{b}{c} = \frac{a - b}{c}$ | $\frac{5}{7} - \frac{3}{7} = \frac{2}{7}$ |

## Unlike Denominators

| Step | Rule |
|------|------|
| 1. Find LCD | LCD of $b$ and $d$ is LCM$(b, d)$ |
| 2. Convert | $\frac{a}{b} = \frac{a \times (L/b)}{L}$ where $L$ = LCD |
| 3. Add/subtract numerators | $\frac{a'}{L} \pm \frac{b'}{L} = \frac{a' \pm b'}{L}$ |
| 4. Simplify | Divide numerator and denominator by their GCF |

**Example**:

$$\frac{1}{3} + \frac{1}{4} = \frac{4}{12} + \frac{3}{12} = \frac{7}{12}$$

## Mixed Numbers

| Method | Steps |
|--------|-------|
| Separate parts | Add whole numbers and fractions independently; regroup if fraction part is improper |
| Convert first | Convert to improper fractions, then follow unlike-denominator rules |

**Example (separate parts)**:

$$2\frac{1}{3} + 1\frac{3}{4} = (2 + 1) + \left(\frac{1}{3} + \frac{3}{4}\right) = 3 + \frac{13}{12} = 3 + 1\frac{1}{12} = 4\frac{1}{12}$$

## Common Errors

| Error | Description | Correct Approach |
|-------|-------------|-----------------|
| Adding denominators | Writing $\frac{1}{3} + \frac{1}{4} = \frac{2}{7}$ | Find LCD first: $\frac{4}{12} + \frac{3}{12} = \frac{7}{12}$ |
| Forgetting to simplify | Writing $\frac{4}{8}$ as the final answer | Simplify: $\frac{4}{8} = \frac{1}{2}$ |
| Ignoring regrouping | Writing $3 + \frac{13}{12}$ as $3\frac{13}{12}$ | Regroup: $3 + 1\frac{1}{12} = 4\frac{1}{12}$ |
