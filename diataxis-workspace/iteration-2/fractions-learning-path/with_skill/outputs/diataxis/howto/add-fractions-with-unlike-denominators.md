# How to Add Fractions with Unlike Denominators

This guide shows you how to add two or more fractions that have different denominators.

## Prerequisites

You can find the least common denominator (LCD) of two numbers and generate equivalent fractions.

## Steps

1. **Find the LCD of all denominators.** For $\frac{2}{3} + \frac{5}{8}$, the LCD of $3$ and $8$ is $24$.

2. **Convert each fraction to an equivalent fraction with the LCD as denominator.**

    $$\frac{2}{3} = \frac{2 \times 8}{3 \times 8} = \frac{16}{24}$$

    $$\frac{5}{8} = \frac{5 \times 3}{8 \times 3} = \frac{15}{24}$$

3. **Add the numerators. Keep the denominator.**

    $$\frac{16}{24} + \frac{15}{24} = \frac{31}{24}$$

4. **Simplify if possible.** $\frac{31}{24}$ is already in lowest terms (GCF of $31$ and $24$ is $1$).

5. **Convert to a mixed number if desired.**

    $$\frac{31}{24} = 1\frac{7}{24}$$

## Handling mixed numbers

If the addends include mixed numbers, convert them to improper fractions first, then follow the steps above.

$$2\frac{1}{4} + 1\frac{2}{3} = \frac{9}{4} + \frac{5}{3}$$

LCD of $4$ and $3$ is $12$:

$$\frac{27}{12} + \frac{20}{12} = \frac{47}{12} = 3\frac{11}{12}$$
